Langkah 1

Instalasi & Setup

Ikuti panduan berikut untuk menyiapkan lingkungan pengembangan (Development Environment) di laptop Anda.

1. Instalasi Python

Pastikan Anda memiliki Python versi 3.10 atau lebih baru.

  • Unduh di python.org
  • Pastikan centang "Add Python to PATH" saat instalasi.

2. Mengatur Environment Variables

Gunakan API Key DeepSeek untuk mengaktifkan AI.

Buka file .env dan masukkan key Anda:

DEEPSEEK_API_KEY=sk-your-real-key-here

Menjalankan di Windows (PowerShell)

Untuk pengguna Windows, ikuti langkah ini untuk menjalankan server:

# Install Library

pip install -r requirements.txt

# Jalankan Server

python main.py

Jika port 8000 sibuk, pastikan tidak ada terminal lain yang sedang menjalankan aplikasi ini.

Langkah 2

Import Dokumen

Pelajari cara memasukkan data dokumen untuk diolah oleh AI PageIndex.

Upload File (PDF/TXT)

Klik tombol "Input Dokumen" lalu pilih file PDF atau TXT. Backend akan memproses file hIngga ribuan kata secara otomatis menggunakan mesin ChromaDB.

Copy-Paste Teks

Punya teks dari artikel atau website? Cukup tempelkan di area teks yang tersedia dan klik 'Proses Ke Node'. AI akan segera memetakannya.

Apa yang terjadi di balik layar?

1

Dokumen dipecah menjadi bagian-bagian kecil (chunking).

2

DeepSeek AI merangkum setiap bab untuk membentuk Hiearchial Tree.

3

Vektor (embedding) disimpan di pangkalan data untuk pencarian semantik kilat.
