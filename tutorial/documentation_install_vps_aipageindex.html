Deployment PageIndex-AI

Panduan teknis langkah demi langkah untuk melakukan instalasi dan deployment aplikasi berbasis Python dengan layanan DeepSeek AI.

1

Mengunduh Source Code (Clone Repository)

Langkah pertama adalah mengambil salinan kode program dari GitHub ke server atau komputer lokalmu.
Perbaikan: Menggunakan git clone untuk mengambil data, bukan git init.

Terminal 
git clone https://github.com/unobengkel/aipageindex.git
cd aipageindex

Penjelasan: Perintah ini akan membuat folder bernama aipageindex yang berisi seluruh logika aplikasi, lalu perintah cd (change directory) digunakan untuk masuk ke dalam folder tersebut.

2

Membuat Lingkungan Terisolasi (Virtual Environment)

Agar library aplikasi ini tidak bentrok dengan aplikasi Python lain di komputermu, kita harus membuat "wadah" khusus.

Terminal 
python3 -m venv venv
source venv/bin/activate

Penjelasan: python3 -m venv venv membuat folder lingkungan virtual bernama venv. Perintah source digunakan untuk "masuk" ke dalam wadah tersebut. Setelah aktif, terminalmu biasanya akan muncul tanda (venv) di bagian depan.

3

Instalasi Dependensi

Aplikasi membutuhkan berbagai pustaka (libraries) pendukung agar bisa berjalan.

Terminal 
pip install -r requirements.txt

Penjelasan: File requirements.txt berisi daftar semua pustaka yang diperlukan (seperti FastAPI, Uvicorn, atau OpenAI/DeepSeek SDK). Perintah ini akan mengunduh dan memasang semuanya secara otomatis ke dalam folder venv.

4

Konfigurasi API Key DeepSeek

Ini adalah bagian krusial agar AI bisa bekerja. Kamu perlu mendaftarkan identitas (API Key) milikmu.

Perintah: nano .env

Isi File .env 
# Format:
DEEPSEEK_API_KEY=sk-isi-key-anda-disini

# Contoh Real:
DEEPSEEK_API_KEY=sk-1232138y218327472164422

Penjelasan: Kita menggunakan file .env (environment variables) agar kunci rahasia ini tidak tertulis langsung di dalam kode program. Ini adalah praktik keamanan standar.

5

Pengaturan Port dan Akses Jaringan

Agar aplikasi bisa diakses dari luar (melalui IP jaringan), kita perlu mengatur alamat host dan port-nya.

Perintah: nano main.py

Edit baris di main.py 
uvicorn.run(app, host="0.0.0.0", port=8000)

Penjelasan:

  • host="0.0.0.0": Artinya aplikasi akan mendengarkan koneksi dari semua alamat IP (tidak hanya localhost).
  • port=8000: Alamat pintu masuk aplikasi. Kamu bisa mengubahnya ke port lain jika port 8000 sudah digunakan.
6

Pengujian Tahap Awal

Sebelum dijalankan secara permanen, kita tes dulu apakah aplikasi berfungsi dengan baik.

Terminal 
python main.py

Cara Cek: Buka browser dan ketikkan alamat IP servermu beserta port-nya. Contoh: http://192.168.1.10:8000 atau http://localhost:8000. Jika muncul tampilan web atau respon JSON, berarti konfigurasi sudah benar.

7

Menjalankan di Background dengan PM2

Jika kamu menutup terminal pada langkah ke-6, aplikasi akan mati. Supaya aplikasi tetap hidup selamanya (bahkan setelah server restart), kita gunakan PM2 (Project Manager).

Terminal 
deactivate
pm2 start main.py --name "PageIndex-AI" --interpreter ./venv/bin/python

Penjelasan:

  • deactivate: Keluar dari virtual environment di sesi terminal saat ini.
  • pm2 start: Memulai aplikasi di latar belakang.
  • --name "PageIndex-AI": Memberikan nama pada proses agar mudah dipantau di list PM2.
  • --interpreter ./venv/bin/python: Memberitahu PM2 untuk menjalankan script menggunakan Python yang ada di dalam folder venv tadi, bukan Python global dari sistem.

Tips Tambahan

Untuk melihat status aplikasi yang sedang berjalan di PM2, kamu bisa menggunakan perintah pm2 status.
Jika ingin melihat log aktivitas atau error, gunakan perintah pm2 logs PageIndex-AI.
